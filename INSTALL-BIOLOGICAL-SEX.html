-- 🧬 Lägg till stöd för biologiskt kön i user_profiles
-- Datum: 2026-03-15
-- Syfte: Aktivera könsbaserade träningsanpassningar

-- 1. Lägg till kolumnen biological_sex
ALTER TABLE user_profiles
ADD COLUMN IF NOT EXISTS biological_sex TEXT;

-- 2. Lägg till validering (endast Man/Kvinna/Annan tillåtna)
ALTER TABLE user_profiles
ADD CONSTRAINT biological_sex_check
CHECK (biological_sex IN ('Man', 'Kvinna', 'Annan') OR biological_sex IS NULL);

-- 3. Kommentar för dokumentation

1. Gå tillbaka till Gymapp
2. Öppna Profil
3. Välj Biologiskt Kön (Man/Kvinna/Annan)
4. Spara profilen ✅

✅ När migrationen är klar:

Män: 72h återhämtning, standard volym (4 set Tier 1)
Kvinnor: 54h återhämtning, +1 set extra (5 set Tier 1)
Annan: 60h återhämtning, +1 set extra (5 set Tier 1)

📚 Vad händer bakom kulisserna?

1. Återhämtningstid:
Kvinnor återhämtar sig 25% snabbare (54h vs 72h) pga lägre testosteronnivåer och bättre kapillärtäthet.

2. Volymtolerans:
Kvinnor tål mer volym (fler set) pga högre andel typ I-muskelfibrer. De får därför +1 set per övning.

3. Strain Score:
Kvinnor får högre Strain Score för samma pass (eftersom de gör fler set). Divisorn är 250 vs mäns 300.

🔍 Felsökning

Problem: Får felmeddelande "permission denied"
Lösning: Kontrollera att du är inloggad som admin på Supabase Dashboard
Problem: Kolumnen finns redan
Lösning: SQL-koden använder IF NOT EXISTS, så det är säkert att köra flera gånger
