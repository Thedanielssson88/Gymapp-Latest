## 🔬 Vetenskaplig Bakgrund

### Training Load Formula
Baserad på forskning från:
- Whoop's proprietary strain algorithm
- Athlytic's recovery scoring
- Session RPE (Foster et al., 2001)

### Acute:Chronic Workload Ratio (ACWR)
**Källa:** Gabbett, T. J. (2016). "The training-injury prevention paradox"

**Fynd:**
- ACWR 0.8-1.3 = Lägst skaderisk
- ACWR >1.5 = 2-4x högre skaderisk
- ACWR <0.8 = Ökad skaderisk (underträning)

**MorphFit Implementation:**
- Använder 7:28 dagars ratio (standard)
- Räknar daily average (inte weekly total)
- Viktar RPE + Duration + Volume

### Muscle Recovery
Baserat på:
- Decay function: Äldre träning = mindre påverkan
- RPE-viktning: Hårdare pass = längre recovery
- Set-volym: Fler sets = mer påverkan

**Formula:**
```
För varje träning inom 7 dagar:
  days_ago = dagar sedan träning
  decay = max(0, 1 - days_ago/7)
  load = sets * (rpe/10) * decay

Total recovery = min(100, sum(load) / 30 * 100)
```

## 🚀 Framtida Förbättringar

### Med Health API Integration:
1. **HRV (Heart Rate Variability)**
   - Lägg till som 4:e faktor (15%)
   - Hög HRV → Bättre återhämtning

2. **Resting Heart Rate**
   - Öka loadScore om RHR är förhöjd
   - Detektera överträning/sjukdom

3. **Sleep Score**
   - Lägg till som 5:e faktor (20%)
   - Dålig sömn → Sänk recovery

**Ny formel med Health Data:**
```
Recovery Index =
  muscleScore * 35% +
  loadScore * 20% +
  timeScore * 15% +
  hrvScore * 15% +
  sleepScore * 15%
```
